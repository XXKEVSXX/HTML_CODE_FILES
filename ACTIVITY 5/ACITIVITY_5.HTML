COMPUTER DICTIONARY

R

Record

a of a file that store data about a particular entity; all records in the file stores the same type of data

Random- access memory (RAM)

the part of the computer’s memory that is not erased when the power is turned off

Resolution

a measurement of the clarity of an image based on the number of pixel used to create the image.

Requirements specification

a list of the necessary hardware and software to put a solution into practice and a list of the wishes of the proposed user of the solution.

Raid

Stands for "Redundant Array of Independent Disks." RAID is a method of storing data on multiple hard disks.

S

Scanner

a device that; allows an image that already exists to be input to the computer so that it can be manipulated using special software

Security of data

the way data is looked after to make sure that it is not damaged, lost or destroyed, typically by making back-ups of the data.

System software

Any program that control the programmer violates the grammatical rules of the programming language.

Syntax: error

An error that Occurs when the programmer violets the grammatical rules of the programming language.

Sequential medium

a storage medium, such as magnetic tapes, the store items one of after the other some sort of order.

T

Transistor

A type of electronic circuitry that controls current flow without the use of a vacuum; it Is smaller, faster, and more reliable than vacuum tubes.

Third - generation computer

Computer developed in the period 1964-1971, they featured integrated circuits, reduced size, lower cost, and increased speed and reliability.

Technical documentation

a set of detailed descriptions about how a solution was arrived at and how it works; it is intended for

Type over mode

A text editing feature in which text that is currently on the screen will be overwritten by the new text you enter.

Touch-sensitive screen

a screen that allow input as well output; input is accomplished by touching an area of the screen

U

User documentation

a set of detailed description about how to use a system to do something useful.

User ID

a unique name or code used to let a computer system know who the user is

User

Anyone who uses the computer.

Uniprocessor

A computer with a single central processing unit, in contrast to a parallel processor. Most personal computers are currently (March 1997) uniprocessors. Some more expensive computers, typically servers, have multiple processors to provide increased throughput.

Uninstaller

A utility program to remove another application program from a computer's disks.

V

Verification

a the checking of data input to a system to ensure that it is what is meant to have been input

Video digitize

a device that turns an analogue picture into a digital one suitable for storing in a computer

Virus

a computer program that is maliciously placed on a computer system with the aim of destroying the files on that system and replicating itself so that system and replicating itself so that it can be transmitted to other systems.

Viruses’ protection

software that identifies viruses and deals with them by not allowing access or erasing them.

Validation

the checking of data input to a system to ensure that it follows certain rules and is therefore sensible.
